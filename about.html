About Send Direct

Send Direct is a peer-to-peer file sharing application that enables secure, direct file transfers between devices without storing your data on any server.

How It Works

Using WebRTC technology, Send Direct establishes a direct connection between the sender and receiver. Your files travel directly from one device to another, encrypted with DTLS 1.3, ensuring maximum privacy and speed.

Key Features

  • No file size limits - share files of any size
  • End-to-end encryption with DTLS 1.3
  • No server storage - files go directly between devices
  • No account required - just share and receive
  • Works across all modern browsers
  • Real-time transfer progress tracking

Technology

Send Direct is built with modern web technologies:

WebRTC DTLS 1.3 WebSocket JavaScript
